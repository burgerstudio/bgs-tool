Termux Ubuntu 容器安装脚本

此脚本可帮助您在Termux上安装 Ubuntu 容器，原脚本来自AnLinux。
相对于原版的变化：所有的raw.githubusercontent.com被raw.kgithub.com替代，解决国内网络无法连接的问题；使用中科大源作为软件源，安装软件更快速；桌面环境（XFCE）增加中文支持（文泉驿-微米黑），解决中文乱码问题；默认挂载手机内部存储为 /sdcard，传输文件更便捷。
启动 Termux，并依次执行以下指令。

1. 安装基本系统
keyboard_arrow_down
pkg install wget openssl-tool proot -y && hash -r && wget https://raw.kgithub.com/burgerstudio/burgerstudio.github.io/main/files/ubuntu.sh && bash ubuntu.sh
2. 换源并启动容器
keyboard_arrow_down
wget https://raw.kgithub.com/burgerstudio/burgerstudio.github.io/main/files/sources.list && mv sources.list ubuntu-fs/etc/apt/ && ./start-ubuntu.sh
3. 安装更新和桌面环境
keyboard_arrow_down
wget https://raw.kgithub.com/burgerstudio/burgerstudio.github.io/main/files/de-apt-xfce4.sh && bash de-apt-xfce4.sh

如果您不需要桌面环境，则只需执行apt update -y。

4. 安装中文字体
keyboard_arrow_down
apt-get install ttf-wqy-microhei -y

如果您未安装桌面环境，则忽略此步骤。
